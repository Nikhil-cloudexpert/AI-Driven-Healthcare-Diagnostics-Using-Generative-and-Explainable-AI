# AI-Driven Healthcare Diagnostics (MERN + Generative AI)

## Overview

This is a full-stack AI-powered healthcare platform built using the MERN stack.
It provides intelligent diagnosis assistance, personalized meal plans, and health insights using Generative AI.

---

## Features

* AI-based Symptom Checker
* AI Nutrition and Meal Planning
* Health Dashboard and Tracking
* JWT-based Authentication
* Real-time AI responses

---

## Tech Stack

* Frontend: React (Vite)
* Backend: Node.js, Express
* Database: MongoDB Atlas
* AI Integration: Gemini API

---

## Setup Instructions

### Backend

cd backend
npm install

Create a `.env` file with:
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret
GEMINI_API_KEY=your_api_key

Run:
npm start

---

### Frontend

cd frontend
npm install
npm run dev
